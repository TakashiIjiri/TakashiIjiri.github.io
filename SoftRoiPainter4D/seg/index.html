領域分割 / Segmentation

Roipainter4Dでは，以下のSegmantationツールを用いて4次元CT画像を領域分割することが可能です． Segmentation toolは，対象領域の特徴にあわせてうまく選択する必要が有ります．
  • 1) Seg Paint and Lasso : voxelを直接ペイントしたり，lassoで囲む事で領域を分割するツールです．手間と時間がかかるので，他のツールではどうしてもうまく分割できない領域に利用すると良いです．
  • 2) Seg Threshold and Region grow : 閾値処理や，閾値を利用した領域拡張法により領域を分割するツールです．他の領域と輝度値の差が大きい領域（骨や造影剤など）の分割に利用できます．
  • 3) Seg Rigid Tracking(ICP) : 剛体追跡（Rigid Tracking）により分割するツールです． シードとなる領域形状Aと等値面の閾値を与えると，その領域を等値面にフィットするように剛体変換する事で全てのフレームの分割が可能です． 剛体運動する領域（骨など）の分割に利用できます．
  • 4) Seg Local Region growing (shere) : 球体状のシードを複数配置すると，そのシード領域内のみで領域拡張が行なわれるツールです． これにより，輝度値にむらのある領域でもうまく分割できる事が有ります．
  • 5) Seg Local Region growing (cylinder) : 円筒状のシードを複数配置すると，そのシード領域内のみで領域拡張が行なわれるツールです． これにより，輝度値にむらのある領域でもうまく分割できる事が有ります．（近々，研究発表を予定してます）
  • 6) Parallel Contors : 平行平面状に輪郭を描くとその輪郭を通るよう二領域境界が生成されるツールです． 円筒に近い形状で，領域境界が曖昧な領域の分割に利用できます．

領域分割のワークフロー

  1. ツールの起動 : メニューバーの『ModeSwitch > seg_*toolname*』をクリックすると領域分割を開始できます．
  2. 領域分割作業 : 各分割ツールを利用して領域分割を行ないます．
  3. 領域登録 : 各分割ツールでは右下にダイアログが表示されます．このダイアログの下にある『finish and store』ボタンをクリックするとその領域とマスクとして登録できます． 登録したマスクにはランダムな色が自動で割り当てられます．
  4. 領域確認 : visualization mask モードでマスクの状態を観察したり，mesh modelとして出力したりすることが出来ます．
