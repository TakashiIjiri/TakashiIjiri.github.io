Refinement in RoiPainter4D

RoiPainter4Dでは，Refiment Stroke Trimモードが使用可能です．
起動 : Click "Menu > mode switch > Ref Stroke Trim"


Refinement Stroke Trim

トリミングした領域を別領域にすることができます．


1) Select Target MaskID
　 d1. Mask list トリムしたいマスクのIDを指定
d2. OK button 決定する
d3. CANCEL button キャンセルする


2) Stroke Trim

　 Shift key + L-drag トリムするマスク領域を指定
Ctrl key + Z key １つ前の状態に戻る
R Click 取り消し


3) Finish and Store

Finish and Store トリムを完了する．
Cancel トリム結果を破棄してStroke Trimモードを終了する
