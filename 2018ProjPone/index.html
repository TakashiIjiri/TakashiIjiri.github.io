Data : Original Data


Each folder contains the following files
1. "*.traw3D_ss" : CT volume (original format, see here for detail)
2. "*.msk" : mask data obtained by segmenting the CT volume
3. "*.obj" : surface model obtained from the CT volumes
4. "*.notFill.bmp" : texture atlas with holes caused by occlusions
5. "*.fill.bmp" : texture atlas of which holes were filled by texture sythesis
6. "photo" folder contains original photographs and their estimated camera position information.
